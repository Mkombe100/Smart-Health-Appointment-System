# Doctor-Patients-Appointment-System

A comprehensive healthcare appointment management system designed to streamline patient scheduling, doctor availability management, and appointment tracking for healthcare facilities.

## Features

- Patient Management and registration
- Doctor scheduling and availability management
- Easy appointment booking system
- Real-time appointment tracking
- Automated notifications and reminders
- Comprehensive dashboard with analytics
- Secure user authentication for patients and providers
- Advanced search and filtering capabilities

## Usage

For Patients:

- Create an account and log in
- Browse available doctors by specialty
- View doctor availability calendar
- Book appointments at preferred date and time
- Receive confirmation notifications
- Manage existing appointments, reschedule or cancel as needed

For Healthcare Providers:

- Access provider dashboard with your credentials
- Set and manage your working hours and availability
- View all scheduled appointments
- Access patient medical history and records
- Generate reports and analyze appointment data

## Project Structure

The project is organized with source code in the src folder containing controllers, models, routes, middleware and utilities. Public assets and views are in their respective folders. Tests are included for quality assurance.
